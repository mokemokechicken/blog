This blog is about the “first steps” and “introduction” that I have made. I often search the web wondering what it’s like, if it’s going to be a hassle, if it’s going to cost money, if I can try it out, and so on.

That’s 10% of my motivation.

The other 20% is because I was curious about how much advertising income there is from blogs.

The remaining 70% is because I want to donate the money I earn (if I can earn it) as a way to repay the fortune of being in a position where I might be able to earn advertising income (I’m serious). I’m hoping to give back to the world a little bit of the booming IT industry.
